# Instrucciones del Proyecto: New Webs

Estas reglas se deben seguir estrictamente para todo lo desarrollado dentro de la carpeta `New Webs`.

## Reglas de Desarrollo:

1.  **Secciones Aceptadas:** Una vez que el usuario confirme que una sección está "Aceptada", **NO se debe volver a modificar**, a menos que se solicite explícitamente un cambio en esa sección específica.
2.  **Comentarios en Español:** Todas las funcionalidades deben tener comentarios descriptivos en español dentro del código (`HTML`, `CSS`, `JS`).
3.  **Separadores de Secciones:** Las separaciones entre secciones del código deben estar etiquetadas claramente en español (ej: `/* ----- INICIO SECCIÓN SERVICIOS ----- */`).
4.  **Codificación:** Mantener un estilo de código limpio, profesional y de alta calidad estética (Rich Aesthetics).
